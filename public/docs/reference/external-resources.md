## Tools & Dependencies

### Docker Desktop

Required for running task environments.

**Download:** [docker.com/products/docker-desktop](https://www.docker.com/products/docker-desktop/)

### uv

Fast Python package manager used for running commands.

**Install:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Documentation:** [docs.astral.sh/uv](https://docs.astral.sh/uv/)

### Ruff

Python linter used for code quality checks.

**Documentation:** [docs.astral.sh/ruff](https://docs.astral.sh/ruff/)

### Git

Version control for GitHub workflow.

**Download:** [git-scm.com](https://git-scm.com/)
